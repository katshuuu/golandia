Урок 89: Выбор по умолчанию в select

Случай `default` в `select` запускается, если другой вариант не готов.

Используйте вариант «по умолчанию», чтобы попробовать отправить или получить без блокировки:

выберите { случай я := <-c: // используем я по умолчанию: // получение от c заблокирует }

Пример для подростков: Это как чат с друзьями: все пишут одновременно, но сообщения нужно обработать без хаоса.

Немного юмора: Горутины лёгкие, но без синхронизации быстро начинается история "кто сломал shared state".
