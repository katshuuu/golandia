Урок 73: Интерфейс Stringer

Одним из наиболее распространенных интерфейсов является [[/pkg/fmt/#Stringer][`Stringer`]], определенный пакетом [[/pkg/fmt/][`fmt`]].

тип Стрингерный интерфейс { Строка() строка }

«Стрингер» — это тип, который может описывать себя как строка. Пакет `fmt` (и многие другие) ищут этот интерфейс для печати значений.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
