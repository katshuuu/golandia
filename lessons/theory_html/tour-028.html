Урок 28: Порядок вычислений в switch

Кейсы переключения оценивают кейсы сверху вниз и останавливаются, когда кейс успешен.

(Например,

переключатель я { случай 0: случай f(): }

не вызывает `f`, если `i==0`.)

#appengine: *Примечание:* Время на игровой площадке Go всегда начинается с #appengine: 2009-11-10 23:00:00 UTC, значение, значение которого оставлено как #appengine: упражнение для читателя.

Пример для подростков: Логика здесь как выбор одежды: если дождь — берёшь зонт, иначе идёшь в кроссах.

Немного юмора: switch удобен, когда вариантов много: мозг говорит "спасибо", а if-else-поездка становится короче.
