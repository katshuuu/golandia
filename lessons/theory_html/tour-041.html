Урок 41: Значения по умолчанию для срезов

При нарезке вы можете опустить верхнюю или нижнюю границы, чтобы использовать вместо них значения по умолчанию.

Значение по умолчанию равно нулю для нижней границы и длине среза для верхней границы.

Для массива

вар а [10]int

эти выражения среза эквивалентны:

а[0:10] а[:10] а[0:] а[:]

Пример для подростков: Срезы и карты можно представить как список дел и словарь никнеймов друзей в телефоне.
