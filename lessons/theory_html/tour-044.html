Урок 44: Создание среза с помощью make

Срезы можно создавать с помощью встроенной функции make; именно так вы создаете массивы динамического размера.

Функция make выделяет обнулённый массив. и возвращает фрагмент, который ссылается на этот массив:

a := make([]int, 5) // len(a)=5

Чтобы указать емкость, передайте третий аргумент команде make:

b := make([]int, 0, 5) // len(b)=0, cap(b)=5

b = b[:cap(b)] // len(b)=5, cap(b)=5 b = b[1:] // len(b)=4, cap(b)=4

Пример для подростков: Срезы и карты можно представить как список дел и словарь никнеймов друзей в телефоне.

Немного юмора: Если перепутал индекс и ключ — не переживай: это классический квест каждого новичка в Go.
