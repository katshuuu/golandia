Урок 70: Пустой интерфейс

Тип интерфейса, который определяет ноль методов, известен как _empty_interface_:

интерфейс{}

Пустой интерфейс может содержать значения любого типа. (Каждый тип реализует как минимум ноль методов.)

Пустые интерфейсы используются кодом, обрабатывающим значения неизвестного типа. Например, `fmt.Print` принимает любое количество аргументов типа `interface{}`.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
