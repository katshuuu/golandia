Урок 86: Буферизированные каналы

Каналы могут быть _буферизованы_. Укажите длину буфера в качестве второго аргумента make для инициализации буферизованного канала:

ch := make(chan int, 100)

Отправляет в буферизованный блок канала только тогда, когда буфер заполнен. Получает блок, когда буфер пуст.

Измените пример, чтобы переполнить буфер, и посмотрите, что произойдет.

Пример для подростков: Это как чат с друзьями: все пишут одновременно, но сообщения нужно обработать без хаоса.

Немного юмора: Горутины лёгкие, но без синхронизации быстро начинается история "кто сломал shared state".
