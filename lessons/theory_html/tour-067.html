Урок 67: Значения интерфейса

На самом деле значения интерфейса можно рассматривать как кортеж значения и тип бетона:

(значение, тип)

Значение интерфейса содержит значение определенного базового конкретного типа.

Вызов метода для значения интерфейса выполняет одноименный метод для значения интерфейса. его основной тип.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
