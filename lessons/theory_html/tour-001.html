Добро пожаловать в Goландию!

Это учебная платформа с AI-репетитором, который проведёт для тебя захватывающую экскурсию по языку Go. Для каждого задания есть интерактивная страница с кодом, чтобы закрепить изученное на практике, а Гоша, живущий на этой платформе всегда готов тебе помочь в любых вопросах⚡️

Наша платформа интерактивная: к каждому заданию есть интерактивная страница с кодом, чтобы закрепить изученное на практике. Для этого перейди на страницу задания и нажми кнопку «Запустить», чтобы скомпилировать и запустить программу в редакторе кода. Результат отобразится ниже кода после автоматической проверки. Ты можешь редактировать программу и запускать её снова, пока не получишь правильный результат ;)
