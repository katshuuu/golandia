Урок 77: Интерфейс Reader

Пакет `io` определяет интерфейс `io.Reader`, который представляет конец чтения потока данных.

Стандартная библиотека Go содержит [[https://cs.opensource.google/search?q=Read%5C(%5Cw%2B%5Cs%5C%5B%5C%5Dbyte%5C)&ss=go%2Fgo][множество реализаций]] этого интерфейса, включая файлы, сетевые подключения, компрессоры, шифры и другие.

Интерфейс `io.Reader` имеет метод `Read`:

func (T) Read(b []byte) (n int, ошибка ошибки)

`Read` заполняет заданный байтовый фрагмент данными и возвращает количество байтов. заполнено и значение ошибки. Он возвращает ошибку `io.EOF`, когда поток заканчивается.

В примере кода создается [[/pkg/strings/#Reader][`strings.Reader`]] и потребляет свои выходные данные по 8 байтов за раз.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
