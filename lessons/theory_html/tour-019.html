Урок 19: Цикл for

В Go есть только одна конструкция цикла — цикл for.

Базовый цикл for состоит из трех компонентов, разделенных точкой с запятой:

- оператор init: выполняется перед первой итерацией - выражение условия: оценивается перед каждой итерацией - оператор post: выполняется в конце каждой итерации

Оператор init часто представляет собой короткое объявление переменной, а объявленные там переменные видны только в области действия `for` заявление.

Цикл прекратит выполнение итерации, как только логическое условие примет значение «ложь».

*Примечание:* В отличие от других языков, таких как C, Java или JavaScript, здесь нет круглых скобок. окружающие три компонента оператора `for` и фигурные скобки `{`}` требуется всегда.

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.

Немного юмора: Маленькие шаги в Go работают лучше всего: запустил, проверил, улучшил — и без паники.
