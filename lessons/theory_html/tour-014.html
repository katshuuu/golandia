Урок 14: Нулевые значения

Переменным, объявленным без явного начального значения, присваиваются их значения. _нулевое_значение_.

Нулевое значение:

- `0` для числовых типов, - `false` для логического типа и - `""` (пустая строка) для строк.

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.

Немного юмора: Маленькие шаги в Go работают лучше всего: запустил, проверил, улучшил — и без паники.
