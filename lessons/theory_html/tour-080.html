Урок 80: Упражнение: Изображения

Помните [[/tour/moretypes/18][генератор изображений]], который вы написали ранее? Давайте напишем еще один, но на этот раз он будет возвращать реализацию image.Image вместо фрагмента данных.

Определите свой собственный тип `Image`, реализуйте [[/pkg/image/#Image][необходимые методы]] и вызовите `pic.ShowImage`.

`Bounds` должен возвращать `image.Rectangle`, например `image.Rect(0,`0,`w,`h)`.

ColorModel должен возвращать color.RGBAModel.

`At` должен возвращать цвет; значение `v` в последнем генераторе изображений соответствует `color.RGBA{v,`v,`255,`255}` в этом.

Пример для подростков: Обобщения — это один и тот же шаблон для разных типов, как универсальный чехол с регулировкой.

Немного юмора: Пишешь функцию один раз, а она работает для int и string — экономия нервов и клавиш.
