Урок 69: Нулевые значения интерфейса

Значение nil интерфейса не содержит ни значения, ни конкретного типа.

Вызов метода на нулевом интерфейсе является ошибкой времени выполнения, поскольку нет введите в кортеж интерфейса, чтобы указать, какой _конкретный_ метод вызывать.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
