Урок 88: Оператор select

Оператор select позволяет горутине ожидать выполнения нескольких коммуникационных операций.

`select` блокируется до тех пор, пока не будет запущен один из его вариантов, а затем он выполняет этот случай. Он выбирает один случайным образом, если готовы несколько.

Пример для подростков: Это как чат с друзьями: все пишут одновременно, но сообщения нужно обработать без хаоса.

Немного юмора: Горутины лёгкие, но без синхронизации быстро начинается история "кто сломал shared state".
