Урок 61: Получатели-указатели

Вы можете объявлять методы с помощью приемников указателей.

Это означает, что тип получателя имеет буквальный синтаксис `*T` для некоторого типа `T`. (Кроме того, `T` сам по себе не может быть указателем, например `*int`.)

Например, здесь метод Scale определен для *Vertex.

Методы с получателями указателей могут изменять значение, на которое получатель точек (как здесь это делает Scale). Поскольку методам часто требуется модифицировать свой получатель, приемники указателей более удобны. распространены, чем получатели ценностей.

Попробуйте удалить `*` из объявления функции Scale в строке 16. и наблюдать, как меняется поведение программы.

При использовании приемника значений метод Scale работает с копией оригинала. Значение `Vertex`. (Это такое же поведение, как и для любого другого аргумента функции.) Метод Scale должен иметь приемник указателя для изменения значения Vertex. объявлено в функции `main`.

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
