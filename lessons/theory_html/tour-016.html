Урок 16: Вывод типов

При объявлении переменной без указания явного типа (с использованием синтаксиса `:=` или синтаксиса выражения `var`=`) тип переменной выводится из значения в правой части.

Когда вводится правая часть объявления, новая переменная имеет тот же тип:

вар я int j := i // j — целое число

Но когда правая часть содержит нетипизированную числовую константу, новая переменная может быть типа int, float64 или complex128, в зависимости от точности константы:

я := 42 // целое f := 3.142 // float64 г := 0,867 + 0,5i // комплекс128

Попробуйте изменить начальное значение `v` в примере кода и посмотрите, как это повлияет на его тип.

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.
