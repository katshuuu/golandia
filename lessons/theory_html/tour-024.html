Урок 24: If с коротким утверждением

Как и оператор for, оператор if может начинаться с короткого оператора, который выполняется перед условием.

Переменные, объявленные в инструкции, находятся в области видимости только до конца if.

(Попробуйте использовать `v` в последнем операторе `return`.)

Пример для подростков: Логика здесь как выбор одежды: если дождь — берёшь зонт, иначе идёшь в кроссах.

Немного юмора: switch удобен, когда вариантов много: мозг говорит "спасибо", а if-else-поездка становится короче.
