Урок 65: Интерфейсы

_interface_type_ определяется как набор сигнатур метода.

Значение типа интерфейса может содержать любое значение, реализующее эти методы.

*Примечание:* В примере кода в строке 22 имеется ошибка. `Vertex` (тип значения) не реализует `Abser`, потому что метод Abs определен только для *Vertex (тип указателя).

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
