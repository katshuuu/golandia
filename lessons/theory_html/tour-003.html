Урок 3: Пакеты

Каждая программа Go состоит из пакетов.

Программы запускаются в пакете `main`.

Эта программа использует пакеты с путями импорта `"fmt"` и `"math/rand"`.

По соглашению имя пакета совпадает с именем последнего элемента пути импорта. Например, пакет math/rand содержит файлы, которые начинаются с оператора packagerand.

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.

Немного юмора: Маленькие шаги в Go работают лучше всего: запустил, проверил, улучшил — и без паники.
