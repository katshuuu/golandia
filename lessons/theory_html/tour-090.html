Урок 90: Упражнение: Эквивалентные бинарные деревья

Может быть много разных двоичных деревьев с одной и той же последовательностью хранящихся в них значений. Например, вот два двоичных дерева, хранящие последовательность 1, 1, 2, 3, 5, 8, 13.

.image /tour/static/img/tree.png

Функция проверки того, хранят ли два двоичных дерева одну и ту же последовательность, в большинстве языков довольно сложна. Мы воспользуемся параллелизмом и каналами Go, чтобы написать простое решение.

This example uses the `tree` package, which defines the type:

type Tree struct { Left *Tree Value int Right *Tree }

Продолжить описание на [[javascript:click('.next-page')][следующая страница]].

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.

Немного юмора: Маленькие шаги в Go работают лучше всего: запустил, проверил, улучшил — и без паники.
