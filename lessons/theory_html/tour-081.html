Урок 81: Параметры типа

Функции Go можно написать для работы с несколькими типами, используя параметры типа. параметры типа функции появляются в квадратных скобках перед именем функции. аргументы.

func Index[T сопоставимый](s []T, x T) int

Это объявление означает, что `s` является срезом любого типа `T`, который удовлетворяет встроенное ограничение `сопоставимо`. `x` также является значением того же типа.

`comparable` — это полезное ограничение, которое позволяет использовать `==` и Операторы `!=` для значений типа. В этом примере мы используем его для сравнения значение для всех элементов среза, пока не будет найдено совпадение. Эта функция `Index` работает для любого типа, поддерживающего сравнение.

Пример для подростков: Обобщения — это один и тот же шаблон для разных типов, как универсальный чехол с регулировкой.

Немного юмора: Пишешь функцию один раз, а она работает для int и string — экономия нервов и клавиш.
