Урок 85: Каналы

Каналы — это типизированный канал, через который вы можете отправлять и получать значения с помощью оператора канала `<-`.

ch <- v // Отправляем v в канал ch. v := <-ch // Получаем из ch и // присваиваем значение v.

(Данные передаются в направлении стрелки.)

Подобно картам и срезам, каналы необходимо создать перед использованием:

ch := make(chan int)

По умолчанию отправляет и получает блок до тех пор, пока другая сторона не будет готова. Это позволяет горутинам синхронизироваться без явных блокировок или условных переменных.

Код примера суммирует числа в срезе, распределяя работу между двумя горутинами. Как только обе горутины завершат свои вычисления, она вычисляет окончательный результат.

Пример для подростков: Это как чат с друзьями: все пишут одновременно, но сообщения нужно обработать без хаоса.

Немного юмора: Горутины лёгкие, но без синхронизации быстро начинается история "кто сломал shared state".
