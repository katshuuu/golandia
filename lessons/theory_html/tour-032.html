Урок 32: Указатели

В Go есть указатели. Указатель содержит адрес значения в памяти.

Тип `*T` является указателем на значение `T`. Его нулевое значение равно `nil`.

вар п *int

Оператор `&` генерирует указатель на свой операнд.

я := 42 р = &я

Оператор `*` обозначает базовое значение указателя.

fmt.Println(*p) // читаем i через указатель p *p = 21 // устанавливаем i через указатель p

Это известно как «разыменование» или «косвенное указание».

В отличие от C, в Go нет арифметики указателей.

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.

Немного юмора: Маленькие шаги в Go работают лучше всего: запустил, проверил, улучшил — и без паники.
