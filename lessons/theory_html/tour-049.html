Урок 49: Упражнение: Срезы

Реализуйте `Pic`. Он должен возвращать фрагмент длиной `dy`, каждый элемент которого представляет собой фрагмент 8-битных беззнаковых целых чисел `dx`. Когда вы запустите программу, она отобразит ваше изображение, интерпретируя целые числа как значения в оттенках серого (ну, в оттенках синего).

Выбор изображения за вами. Интересные функции включают `(x+y)/2`, `x*y` и `x^y`.

(Вам необходимо использовать цикл для размещения каждого `[]uint8` внутри `[][]uint8`.)

(Используйте uint8(intValue) для преобразования между типами.)

Пример для подростков: Представь, что объясняешь тему другу на перемене: если можешь просто рассказать, ты понял материал.
