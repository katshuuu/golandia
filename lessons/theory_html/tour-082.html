Урок 82: Обобщённые типы

Помимо универсальных функций, Go также поддерживает универсальные типы. Тип может параметризоваться параметром типа, который может быть полезен для реализации общие структуры данных.

В этом примере показано простое объявление типа для односвязного списка. хранящий любой тип значения.

В качестве упражнения добавьте некоторые функциональные возможности в эту реализацию списка.

Пример для подростков: Обобщения — это один и тот же шаблон для разных типов, как универсальный чехол с регулировкой.
