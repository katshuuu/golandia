Урок 60: Методы, продолжение

Вы также можете объявить метод для неструктурных типов.

В этом примере мы видим числовой тип MyFloat с методом Abs.

Вы можете объявить метод только с получателем, тип которого определен в том же файле. package в качестве метода. Вы не можете объявить метод с получателем, тип которого определен в другом пакет (который включает встроенные типы, такие как `int`).

Пример для подростков: Интерфейс — как универсальный порт зарядки: важно, чтобы устройство поддерживало нужный разъём.

Немного юмора: Go не требует слова implements: если методы совпали, тип уже в команде.
